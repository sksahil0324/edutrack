## Project Overview

**EduTrack AI** is an AI-driven web application designed to enhance student success through:
- **Dropout Risk Prediction**: AI-powered risk assessment identifying at-risk students early
- **Gamification**: Motivate students with XP, levels, badges, and challenges
- **Teacher Intervention Tools**: Comprehensive dashboards for monitoring and intervention management
- **Admin Oversight**: System management, user administration, and analytics

### Key Features
- Real-time risk assessments using AI algorithms
- Gamified learning with XP, levels, and badges
- Teacher intervention tracking and effectiveness measurement
- Admin dashboard with system statistics and code archiving
- Email OTP authentication with anonymous guest access
- Role-based access control (Student, Teacher, Admin)

### Tech Stack
- **Frontend**: React + TypeScript + Tailwind CSS + Shadcn UI
- **Backend**: Convex (serverless database & functions)
- **Authentication**: Convex Auth (Email OTP + Anonymous)
- **Routing**: React Router
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS with custom theme
